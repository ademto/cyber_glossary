## What is a Backup?

A **Backup** is a safety copy of data that can be restored in case of accidental deletion, cyberattack, hardware failure, or natural disaster. It’s a critical part of business continuity planning.

## Backup Types

- **Full** – All data
- **Incremental** – Changes since last backup
- **Differential** – Changes since last full backup

## Best Practices

- Use the 3-2-1 rule
- Automate backups
- Regularly test restore process
- Encrypt and secure backup locations
